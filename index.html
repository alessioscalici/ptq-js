Write a simple arithmetic expression to see the parse tree (e.g. "5 + 2", "x * (9 - y)"...)
-- main
    WHITESPACE  / [:space:]+  / -ignore
    PLUS        / \+ /
    MINUS       / - /
    MULT        / \* /
    DIV         / \/ /
    LEFT_PAR    / \( /
    RIGHT_PAR   / \) /
    NUMBER      / [0-9]+ /
    IDENTIFIER  / [a-zA-Z_][a-zA-Z_0-9]* /
< expr > ::= < term > PLUS < expr >
    | < term > MINUS < expr >
    | < term >
< term > ::= < factor > MULT < term >
    | < factor > DIV < term >
    | < factor >
< factor > ::= LEFT_PAR < expr > RIGHT_PAR
    | NUMBER
    | IDENTIFIER
